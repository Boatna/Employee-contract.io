🔧 วิธีตั้ง Alt Text ใน Google Slides (ทำซ้ำกับทุก Shape)
1

Upload ไฟล์ PPTX ขึ้น Google Drive

ลาก .pptx วางใน Drive → คลิกขวา → Open with Google Slides
หรือสร้าง Slides ใหม่แล้วออกแบบตามต้นฉบับ

2

แก้ข้อความใน TextBox

ดับเบิลคลิก TextBox → ลบข้อความเดิม → พิมพ์ Placeholder เช่น {{AREA}}
สำคัญ: ต้องเหลือเฉพาะ Placeholder เท่านั้น ไม่มีข้อความอื่นปนอยู่

✅ ถูก:  {{AREA}}
❌ ผิด:  พื้นที่ : {{AREA}} (มีข้อความอื่นปน → Script จะหาไม่เจอ)
💡 แก้: ให้แยก TextBox — TextBox นึงพิมพ์ "พื้นที่ :" อีก TextBox พิมพ์ {{AREA}}
3

ตั้ง Alt Text ให้กล่อง Checkbox และรูปภาพ

คลิก Shape/รูปที่ต้องการ → เมนู Format (หรือคลิกขวา) → Alt text
ใส่ Placeholder ใน ช่อง Title (บรรทัดบนสุด) เช่น {{SS5_สะสาง}}

Format → Alt text → Title:  {{SS5_สะสาง}}
ช่อง Description: เว้นว่างได้

⚠️ อย่าใส่ใน Description — Script อ่านแค่ Title เท่านั้น
4

จัดการกล่อง Waste ที่ซ้ำกัน

ในต้นฉบับมีตัวอักษร D, M, E ปรากฏซ้ำ 2 ครั้ง ให้เลือก:

ทางเลือก A — ลบทิ้ง: คลิกเลือก Rectangle ที่ซ้ำ → Delete
ทางเลือก B — ใช้สำหรับกล่องที่หายไป: เปลี่ยนข้อความเป็นตัวอักษรที่ขาด (S, N) แล้วตั้ง Alt Text ให้ถูกต้อง
5

บันทึกเป็น Template และคัดลอก Slide ID

หลังแก้ครบ → URL ของ Presentation จะมีรูปแบบ:
https://docs.google.com/presentation/d/1BxiMVs0.../edit
คัดลอก ID ตรงนั้น → วางใน TEMPLATE_SLIDE_ID ใน Code.gs

6

ทดสอบด้วยการ Submit ฟอร์ม

Deploy Web App → เปิดฟอร์ม → กรอกข้อมูลทดสอบ → กด Submit
ตรวจสอบว่า Placeholder ถูกแทนที่ครบ + กล่อง Checkbox เปลี่ยนเป็นสีเหลือง + รูปแสดงถูกตำแหน่ง
